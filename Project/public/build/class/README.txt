강의실 이미지 파일 명명 규칙

1. 강의실 대표 이미지: [강의실번호].PNG
   예: 7205.PNG, 7305.PNG

2. 강의실 내부 이미지: [강의실번호]_in.jpg
   예: 7205_in.jpg, 7305_in.jpg

3. 강의실 외부 이미지: [강의실번호]_out.jpg
   예: 7205_out.jpg, 7305_out.jpg

강의실 이미지는 building_data.js 파일에서 각 강의실의 roomDetails 객체에 설정되어 있습니다.
이미지 경로는 /build/class/ 폴더를 기준으로 지정됩니다.
